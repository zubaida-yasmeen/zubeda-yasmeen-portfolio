# **App Name**: Zubeda Yasmeen Portfolio

## Core Features:

- Hero Narrative: A high-impact landing section with typing animations to showcase titles like Cloud Engineer and AI Builder.
- Static Experience Timeline: A CSS-powered vertical timeline documenting leadership roles in the AWS Student Builder Group and community initiatives.
- Project Gallery: A grid of project cards featuring hover-lift effects, gradient tech stack badges, and direct GitHub links.
- Performance Stats: Visual data cards highlighting the AWS GenAI Hackathon 3rd place win, CGPA, and community growth metrics.
- Dark Mode Switcher: A lightweight JavaScript toggle to shift between a premium dark theme and a standard high-contrast light theme.
- Optimized Static Delivery: SEO-friendly metadata and pre-rendered content for high speed and mobile responsiveness without backend overhead.

## Style Guidelines:

- Deep midnight background (#0f0f1a) paired with accent colors of Purple (#8b5cf6), Pink (#ec4899), and Green (#22c55e).
- Modern, tech-focused font selection with clear hierarchy for headlines and monospaced accents for technical details.
- Clean, card-based responsive layout using CSS Grid and Flexbox, intentionally avoiding glassmorphism for a solid feel.
- Subtle CSS-only interactions including slide-ups, fade-ins on scroll, and hover-state vertical translations.
- Minimalist SVG icons styled with gradients and simple hover transitions to represent tools like AWS, Linux, and Firebase.
